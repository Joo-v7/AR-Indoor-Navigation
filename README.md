# AR-Indoor-Navigation
Unity(2021.2.3f1)를 활용한 조선대학교 IT융합대학 실내 내비게이션.

# 나의 역할
- 팀장
- 프로토타입 제작
- 피그마를 활용한 화면 설계
- 도착지 설정 및 거리 계산 구현
- 도착 이벤트 구현
